# Art Pack (Original SVG)

These SVGs are original, stylized assets for the MVP.

## Usage
- Import into your SpriteKit asset catalog as vector/PDF or rasterize to PNG.
- Suggested names: player, roach, collectible, ground, stoop, trashbag, car, background.
- The art is intentionally bold and readable at small sizes.
- If Inkscape is installed, rasterized PNG/PDFs are generated in art/rasterized.
